### 1. Orthogonality: The Foundations

- **Orthogonal Vectors**: Two vectors $x$ and $y$ are orthogonal if they are perpendicular, which is mathematically defined by their dot product being zero ($x^T y = 0$).
- **Pythagorean Theorem**: For any two orthogonal vectors, the square of the length of their sum equals the sum of the squares of their lengths: $||x||^2 + ||y||^2 = ||x + y||^2$.
- **Orthogonality of Subspaces**: Two subspaces $V$ and $W$ are orthogonal if every vector in $V$ is perpendicular to every vector in $W$.
    - **Condition for Orthogonality**: If two subspaces intersect at any vector other than the zero vector, they cannot be orthogonal.
    - **Dimension Rule**: Orthogonality between subspaces $V$ and $W$ in a vector space $S$ is only possible if $\dim(V) + \dim(W) \leq \dim(S)$.

### 2. The "Big Picture": Orthogonal Complements

The four fundamental subspaces of a matrix $A$ are paired as orthogonal complements, meaning they are perpendicular and their dimensions fill the entire space:

- **In $R^n$**: The **Row Space** $C(A^T)$ is the orthogonal complement of the **Nullspace** $N(A)$.
- **In $R^m$**: The **Column Space** $C(A)$ is the orthogonal complement of the **Left Nullspace** $N(A^T)$.
- **The Logic**: Every vector $x$ in the nullspace ($Ax=0$) is perpendicular to every row of $A$ because the dot product of each row with $x$ is zero.

### 3. Orthogonal Matrices ($Q$)

- **Definition**: A matrix $Q$ is called an **orthonormal matrix** if its columns $q_1, q_2, \dots, q_n$ are unit vectors ($||q_i||=1$) and are all perpendicular to each other ($q_i^T q_j = 0$ for $i \neq j$).
- **Orthogonal Matrix**: If an orthonormal matrix is square, it is termed an **Orthogonal Matrix**.
- **Key Property**: For any orthogonal matrix $Q$, the transpose is its inverse ($Q^T Q = I \implies Q^T = Q^{-1}$).
- **Projection**: The projection of a vector $b$ onto the column space of an orthogonal matrix $Q$ is simply the vector $b$ itself, as $b$ is already contained within that span.

### 4. The Gram-Schmidt Process

The purpose of this process is to transform a set of linearly independent vectors ($a, b, c, \dots$) into a set of **orthonormal vectors** ($q_1, q_2, q_3, \dots$) that span the same space.

**Step-by-Step Logic**:

1. **Start**: Set the first orthogonal vector $A$ equal to the first original vector $a$.
2. **Orthogonalize $b$**: Subtract the projection of $b$ onto $A$ from $b$ to get the second orthogonal vector $B$: $B = b - \frac{A^T b}{A^T A} A$.
3. **Orthogonalize $c$**: Subtract the projections of $c$ onto both $A$ and $B$: $C = c - \frac{A^T c}{A^T A} A - \frac{B^T c}{B^T B} B$.
4. **Normalize**: Convert $A, B, C$ into unit vectors $q_1, q_2, q_3$ by dividing each by its length.

### 5. Assignment Solution: Question 5.1 (i)*

**Apply Gram-Schmidt for vectors: $a = \begin{bmatrix} 1 & 2 & 0 \end{bmatrix}^T$, $b = \begin{bmatrix} 8 & 1 & -6 \end{bmatrix}^T$, $c = \begin{bmatrix} 0 & 0 & 1 \end{bmatrix}^T$**.

**Step 1: Find Orthogonal Vector $A$**

- $A = a = \mathbf{\begin{bmatrix} 1 \ 2 \ 0 \end{bmatrix}}$

**Step 2: Find Orthogonal Vector $B$**

- $B = b - \frac{A^T b}{A^T A} A$
- $A^T b = (1)(8) + (2)(1) + (0)(-6) = 10$.
- $A^T A = (1)^2 + (2)^2 + (0)^2 = 5$.
- $B = \begin{bmatrix} 8 \ 1 \ -6 \end{bmatrix} - \frac{10}{5} \begin{bmatrix} 1 \ 2 \ 0 \end{bmatrix} = \begin{bmatrix} 8 \ 1 \ -6 \end{bmatrix} - \begin{bmatrix} 2 \ 4 \ 0 \end{bmatrix} = \mathbf{\begin{bmatrix} 6 \ -3 \ -6 \end{bmatrix}}$

**Step 3: Find Orthogonal Vector $C$**

- $C = c - \frac{A^T c}{A^T A} A - \frac{B^T c}{B^T B} B$
- $A^T c = (1)(0) + (2)(0) + (0)(1) = 0$.
- $B^T c = (6)(0) + (-3)(0) + (-6)(1) = -6$.
- $B^T B = (6)^2 + (-3)^2 + (-6)^2 = 36 + 9 + 36 = 81$.
- $C = \begin{bmatrix} 0 \ 0 \ 1 \end{bmatrix} - \frac{0}{5} A - \frac{-6}{81} \begin{bmatrix} 6 \ -3 \ -6 \end{bmatrix} = \begin{bmatrix} 0 \ 0 \ 1 \end{bmatrix} + \frac{2}{27} \begin{bmatrix} 6 \ -3 \ -6 \end{bmatrix} = \mathbf{\begin{bmatrix} 4/9 \ -2/9 \ 5/9 \end{bmatrix}}$

**Step 4: Normalize to Orthonormal Vectors ($q_i$)**

- $q_1 = \frac{A}{||A||} = \mathbf{\frac{1}{\sqrt{5}} \begin{bmatrix} 1 \ 2 \ 0 \end{bmatrix}}$
- $q_2 = \frac{B}{||B||} = \frac{1}{\sqrt{81}} \begin{bmatrix} 6 \ -3 \ -6 \end{bmatrix} = \mathbf{\begin{bmatrix} 2/3 \ -1/3 \ -2/3 \end{bmatrix}}$
- $q_3 = \frac{C}{||C||} = \dots \text{Calculating length of } C: \sqrt{(4/9)^2 + (-2/9)^2 + (5/9)^2} = \sqrt{45/81} = \frac{\sqrt{5}}{3}$.
- $q_3 = \frac{3}{\sqrt{5}} \begin{bmatrix} 4/9 \ -2/9 \ 5/9 \end{bmatrix} = \mathbf{\frac{1}{3\sqrt{5}} \begin{bmatrix} 4 \ -2 \ 5 \end{bmatrix}}$

Final orthonormal set: **${q_1, q_2, q_3}$**.
