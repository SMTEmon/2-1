## Question 1: PDA & TM Decidability (30 Marks)

### 1. PDA for $L_1 = \{0^n 1^{2n} \mid n \geq 0\}$
**Logic:** For every `0` read, push two markers (e.g., `XX`) onto the stack. For every `1` read, pop one `X`.
- **Transitions:**
  1. $\delta(q_{start}, \epsilon, \epsilon) \to (q_0, \$)$ (Push bottom marker)
  2. $\delta(q_0, 0, \epsilon) \to (q_0, XX)$ (Read 0, push two X's)
  3. $\delta(q_0, 1, X) \to (q_1, \epsilon)$ (Switch to popping on first 1)
  4. $\delta(q_1, 1, X) \to (q_1, \epsilon)$ (Continue popping)
  5. $\delta(q_1, \epsilon, \$) \to (q_{acc}, \epsilon)$ (Accept if stack is empty)

### 2. PDA for $L_2 = \{a^m b^n c^p \mid m = n \text{ or } n = p, \text{ but not both}\}$
**Logic:** Nondeterministically branch into two paths.
- **Branch 1 ($m=n$ AND $n \neq p$):**
  - Push $a$'s. Pop for $b$'s. Ensure stack is empty after $b$'s.
  - Read $c$'s and use a sub-branch to ensure the count of $c$'s is either strictly less than or strictly greater than the count of $b$'s (this part is tricky and usually requires "guessing" the inequality).
- **Branch 2 ($n=p$ AND $m \neq n$):**
  - Read and ignore $a$'s (ensuring $m \neq n$ via sub-branch).
  - Push $b$'s. Pop for $c$'s. Accept on empty stack.
*Note: The "but not both" constraint is handled by the fact that $a^n b^n c^n$ would satisfy both branches, but since we only need to accept if ONE branch accepts, we must be careful. However, in standard PDA construction for XOR, we typically just build the union and prove the intersection is handled or specifically excluded.*

### 3. $EQ_{DFA}$ is Decidable
**Proof:**
1. On input $\langle A, B \rangle$, where $A, B$ are DFAs:
2. Construct a new DFA $C$ that recognizes the **Symmetric Difference**: $L(C) = (L(A) \cap \overline{L(B)}) \cup (\overline{L(A)} \cap L(B))$.
   - $L(C)$ contains strings that are in $A$ but not $B$, or in $B$ but not $A$.
3. If $L(A) = L(B)$, then $L(C) = \emptyset$.
4. Run the decider for $E_{DFA}$ (Emptiness) on $\langle C \rangle$.
5. If $E_{DFA}$ accepts, **Accept**. Otherwise, **Reject**.

## Question 2: CFG, CNF, CYK & Pumping Lemma (41 Marks)

### 1. Language Interpretation
**Grammar $G$:** $S \to aSa \mid bSb \mid A$, $A \to aA \mid bA \mid \epsilon$.
- **Description:** The language $L(G)$ consists of all strings over $\{a, b\}$ such that the string contains a central part (generated by $A$) wrapped in matching prefixes/suffixes (generated by $S$). Since $A \to \Sigma^*$, $S$ can generate any string by simply reducing to $A$ immediately.
- **Wait, Correction:** $S$ generates strings of the form $w u w^R$ where $w, u \in \{a,b\}^*$. But since $u$ can be any string (including the empty string), any string $x$ can be written as $\epsilon x \epsilon^R$. Thus, $L(G) = \Sigma^*$.
- **Example Strings:** `ab`, `baaba`, `aaa`, `$\epsilon$`.

### 2. CFG to CNF Conversion
**Original:** $S \rightarrow ASA \mid aB$, $A \rightarrow B \mid S$, $B \rightarrow b \mid \epsilon$
1. **New Start:** $S_0 \to S$
2. **Remove $\epsilon$ ($B \to \epsilon$):**
   - $S \to ASA \mid aB \mid a$
   - $A \to B \mid S \mid \epsilon$
3. **Remove $\epsilon$ ($A \to \epsilon$):**
   - $S \to ASA \mid AS \mid SA \mid S \mid aB \mid a$
   - $A \to B \mid S$
4. **Remove Unit Rules:**
   - $S \to ASA \mid AS \mid SA \mid aB \mid a$ (Note: $S \to S$ is discarded)
   - $A \to b \mid ASA \mid AS \mid SA \mid aB \mid a$
   - $S_0 \to ASA \mid AS \mid SA \mid aB \mid a$
5. **Binarize & Terminals:**
   - $S \to AV_1 \mid AS \mid SA \mid U_a B \mid a$
   - $V_1 \to SA$
   - $U_a \to a$
   - (Repeat for $A$ and $S_0$).

### 3. CYK Table for `baaba`
**Grammar:** $S \to AB \mid BC, A \to BA \mid a, B \to CC \mid b, C \to AB \mid a$

| $j \setminus i$ | 1 (`b`) | 2 (`a`) | 3 (`a`) | 4 (`b`) | 5 (`a`) |
| :--- | :--- | :--- | :--- | :--- | :--- |

### 4. Pumping Lemma for $L = \{a^n b^m c^n d^m\}$
1. Assume $L$ is context-free. Let $p$ be the pumping length.
2. Choose $s = a^p b^p c^p d^p \in L$.
3. By $|vxy| \leq p$, $vxy$ can span at most two adjacent symbol blocks (e.g., $a$'s and $b$'s, or $b$'s and $c$'s).
4. **Case 1:** $vxy$ is within one block (e.g., $a$'s). Pumping $v, y$ changes the count of $a$'s but not $c$'s. $n \neq n \implies \notin L$.
5. **Case 2:** $vxy$ spans $a$'s and $b$'s. Pumping increases $a$'s and $b$'s but not $c$'s and $d$'s. $n \neq n$ AND $m \neq m \implies \notin L$.
6. **Case 3:** $vxy$ spans $b$'s and $c$'s. Pumping increases $b$'s and $c$'s. $m \neq m$ AND $n \neq n \implies \notin L$.
7. In all cases, the balance is broken. Contradiction.

## Question 3: TM Variants, Church-Turing & Undecidability (32 Marks)

### 1. NTM Simulation
- **Mechanism:** A DTM simulates an NTM by exploring its computation tree. The DTM uses three tapes: Input tape (read-only), Simulation tape (copies of branches), and Address tape (tracks the current node in the tree using coordinates like `1-2-1`).
- **BFS Requirement:** BFS is used because some branches of an NTM might be infinite. If a DTM uses DFS, it might follow an infinite branch and never find an accepting configuration that exists at a shallow depth in another branch. BFS guarantees finding the shortest accepting path.

### 2. Church-Turing Thesis
- **Thesis:** Any algorithmic process that can be carried out by a human or computer can be performed by a Turing Machine.
- **Justification:** TMs are robust. Variants (multi-tape, NTM, etc.) don't increase power. This allows us to use high-level descriptions because we know they can be translated into valid TM transitions.

### 3. Reduction: $A_{TM} \leq_m HALT_{TM}$
1. Assume $R$ decides $HALT_{TM}$.
2. Build decider $S$ for $A_{TM}$: "On input $\langle M, w \rangle$:
   a. Run $R$ on $\langle M, w \rangle$.
   b. If $R$ rejects (meaning $M$ loops on $w$), **Reject**.
   c. If $R$ accepts (meaning $M$ halts on $w$), simulate $M$ on $w$.
   d. If the simulation accepts, **Accept**; otherwise **Reject**."
3. Since $S$ decides $A_{TM}$ and we know $A_{TM}$ is undecidable, $R$ cannot exist.

### 4. $A_{CFG}$ Decidability
1. **Algorithm:** On input $\langle G, w \rangle$:
   a. Convert $G$ to Chomsky Normal Form (CNF).
   b. Let $n = |w|$. If $n=0$, check if $S \to \epsilon$ is possible.
   c. If $n>0$, list all derivations of length $2n-1$.
   d. If any derivation results in $w$, **Accept**; otherwise **Reject**.
2. **Halting Guarantee:** In CNF, every derivation of a string of length $n$ takes exactly $2n-1$ steps. The number of such derivations is finite, so the TM only checks a finite set and always halts.

## Question 4: True/False with Justification (17 Marks)

1. **True**. CFLs are recognizable by PDAs, and TMs can simulate PDAs. Every CFL has a decider (e.g., CYK algorithm).
2. **False**. CFLs are not closed under intersection. $\{a^n b^n c^m\} \cap \{a^m b^n c^n\} = \{a^n b^n c^n\}$ (not CFL).
3. **True**. If $B$ is decidable and $A$ reduces to it, we can decide $A$ by transforming the input and running $B$'s decider.
4. **False**. This is the **Totality Problem** ($ALL_{TM}$), which is $\Pi_2$-complete and not even Turing-recognizable. It is strictly harder than the Halting Problem.
