## Question 2: CFG, CNF, CYK & Pumping Lemma (41 Marks)

1. **Describe the language** generated by the following CFG in English or set notation. Give 2 example strings that belong to the language. *(8 marks)*
   $$G = (\{S, A\}, \{a, b\}, R, S)$$
   Rules:
   - $S \rightarrow aSa \mid bSb \mid A$
   - $A \rightarrow aA \mid bA \mid \epsilon$

2. **Convert the following CFG to Chomsky Normal Form (CNF).** Show each step clearly. *(10 marks)*
   - $S \rightarrow ASA \mid aB$
   - $A \rightarrow B \mid S$
   - $B \rightarrow b \mid \epsilon$

3. **CYK Algorithm (15 marks)**
   Given the CNF grammar:
   - $S \rightarrow AB \mid BC$
   - $A \rightarrow BA \mid a$
   - $B \rightarrow CC \mid b$
   - $C \rightarrow AB \mid a$
   
   Run the CYK algorithm on the string $w = \texttt{baaba}$. 
   - Draw the complete triangular parsing table.
   - State whether $w \in L(G)$.

4. **Pumping Lemma for CFLs**
   Use the Pumping Lemma to prove that $L = \{a^n b^m c^n d^m \mid n, m \geq 0\}$ is **not** context-free. *(8 marks)*

## Question 3: TM Variants, Church-Turing & Undecidability (32 Marks)

1. **NTM Simulation** *(8 marks)*
   - Explain how a Nondeterministic Turing Machine (NTM) is simulated by a standard deterministic single-tape TM.
   - Why is **Breadth-First Search (BFS)** required instead of Depth-First Search (DFS)?

2. **Church-Turing Thesis** *(6 marks)*
   - State the Church-Turing Thesis.
   - How do we justify using high-level descriptions (e.g., "sort the array", "check if graph is connected") when defining TM algorithms?

3. **Reduction Proof** *(10 marks)*
   Prove that $HALT_{TM}$ is undecidable by **reducing $A_{TM}$ to it**. 
   - Assume $R$ decides $HALT_{TM}$.
   - Construct a decider $S$ for $A_{TM}$ using $R$.
   - Show the contradiction.

4. **TM Construction** *(8 marks)*
   Design a high-level TM that decides $A_{CFG} = \{\langle G, w \rangle \mid G \text{ is a CFG that generates string } w\}$. 
   - Describe the algorithm.
   - Explain why this TM is **guaranteed to halt** on all inputs.

## Question 4: True/False with Justification (17 Marks)

Determine whether each statement is **True** or **False**. Provide a concise 1–2 sentence justification.

1. Every context-free language is Turing-decidable.
2. The class of context-free languages is closed under intersection.
3. If language $A$ reduces to language $B$ ($A \leq_m B$) and $B$ is decidable, then $A$ must be decidable.
4. There exists a Turing machine that accepts exactly those inputs $\langle M \rangle$ where $M$ is a TM that halts on all inputs.

# Answer Key / Hints

> [!hint] Q1 Hints
> 1. **$L_1 = \{0^n 1^{2n}\}$**: Push two markers (e.g., `XX`) for every `0` read. Then pop one `X` for every `1`. Accept if stack empties exactly when input ends.
> 2. **$L_2$**: Use nondeterminism. Branch 1: check $m=n$ and ensure $n \neq p$. Branch 2: check $n=p$ and ensure $m \neq n$. Use stack to count matching pairs, then verify the mismatch by consuming remaining input without matching stack.
> 3. **$EQ_{DFA}$**: Construct DFA $C$ for symmetric difference $L(A) \oplus L(B) = (L(A) \cap \overline{L(B)}) \cup (\overline{L(A)} \cap L(B))$. Run emptiness decider $E_{DFA}$ on $C$. If empty, accept; else reject.

> [!hint] Q2 Hints
> 1. **Language**: $A$ generates $\Sigma^*$, so $S$ generates $\Sigma^* \cup \{\text{even palindromes}\} = \Sigma^*$. Examples: `ab`, `baab`.
> 2. **CNF Steps**:
>    - New start: $S_0 \rightarrow S$
>    - Remove $\epsilon$: $B \rightarrow \epsilon$ affects $S \rightarrow aB \Rightarrow S \rightarrow a$. $A \rightarrow B$ and $S \rightarrow ASA$ need combinations.
>    - Remove unit rules: $A \rightarrow B$, $A \rightarrow S$, etc.
>    - Binarize & replace terminals.
> 3. **CYK Table for `baaba`**: Fill bottom-up. $T[i,1]$ from terminals. $T[i,j]$ from splits $k$. Check if $S \in T[1,5]$.
> 4. **Pumping Lemma**: Choose $s = a^p b^p c^p d^p$. Since $|vxy| \leq p$, $vxy$ spans at most 2 adjacent blocks. Pumping breaks the $n=n$ or $m=m$ balance.

> [!hint] Q3 Hints
> 1. **NTM**: DTM simulates all branches by storing them on tape separated by `#`. BFS ensures we explore all finite-length branches before infinite ones. DFS could get stuck down an infinite rejecting branch and miss an accepting one.
> 2. **Church-Turing**: "Any effective procedure = TM". High-level ops are valid because they can be compiled into finite TM states/transitions. We trust the thesis.
> 3. **Reduction**: Given $R$ for $HALT_{TM}$, build $S$ for $A_{TM}$: Run $R(\langle M,w \rangle)$. If rejects, reject. If accepts, simulate $M$ on $w$ and output result. $S$ always halts $\Rightarrow$ contradiction since $A_{TM}$ undecidable.
> 4. **$A_{CFG}$**: Convert $G$ to CNF. String $w$ of length $n$ has derivations of exactly $2n-1$ steps. Generate all $2n-1$ step derivations (finite number). Check if any yield $w$. Halts because search space is finite.

> [!check] Q4 Answers
> 1. **True**. CFL $\subset$ Decidable. A TM can simulate a PDA or use CYK on CNF.
> 2. **False**. CFLs are not closed under intersection. Counterexample: $\{a^n b^n c^m\} \cap \{a^m b^n c^n\} = \{a^n b^n c^n\}$ (not CFL).
> 3. **True**. If $B$ is decidable, we can decide $A$ by reducing $A \to B$ and running $B$'s decider.
> 4. **False**. This is the "Totality Problem" ($ALL_{TM}$), which is not even Turing-recognizable (strictly harder than $A_{TM}$).
