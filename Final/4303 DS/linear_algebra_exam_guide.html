Lecture 16

Eigenvalues & Eigenvectors

The Core Idea

For a matrix A, an eigenvector x is a special vector that only gets scaled (not rotated) when multiplied by A. The scaling factor is the eigenvalue λ.

Ax = λx ⟹ (A − λI)x = 0

For this to have a non-trivial solution (x ≠ 0), the matrix (A − λI) must be singular:

det(A − λI) = 0 ← the characteristic equation

Step-by-Step: Finding Eigenvalues & Eigenvectors

  1. Write the characteristic equation: det(A − λI) = 0
  2. Expand and solve for λ (you get the eigenvalues)
  3. For each λ, substitute back into (A − λI)x = 0
  4. Row reduce and express the free variable as 1 to get the eigenvector

From the quiz: for a 2×2 matrix [a b; c d], use the shortcut:

λ² − trace(A)·λ + det(A) = 0

Key Properties

  • →An n×n matrix has n eigenvalues (counted with multiplicity, possibly complex)
  • →Product of all eigenvalues = det(A)
  • →Sum of all eigenvalues = trace(A)
  • →If A is triangular, eigenvalues are the diagonal entries

Adding kI to A

If A has eigenvalue λ with eigenvector x, then (A + kI) has eigenvalue λ + k with the same eigenvector x.

(A + kI)x = Ax + kIx = λx + kx = (λ+k)x

The eigenvectors don't change — only eigenvalues shift.

Eigenvalues of Special Matrices

Matrix TypeEigenvaluesEigenvectors
Projection P onto line λ = 1 (in-plane), λ = 0 (perpendicular) Vectors in the plane; vectors ⊥ to plane
Reflection R across plane λ = 1 (in-plane), λ = −1 (normal) Vectors in mirror plane; normal vector
Rotation Q by angle θ λ = e^(iθ), e^(−iθ) — complex unless θ = 0, π No real eigenvectors (except at 0° or 180°)
⚡ Geometric Intuition

A negative eigenvalue means the eigenvector gets flipped (points in the opposite direction). For example, reflection has λ = −1 for the normal vector — multiplying by R reverses it.

Not All Matrices Have n Independent Eigenvectors

When an eigenvalue is repeated (algebraic multiplicity > 1), there may be fewer independent eigenvectors than expected. This is the case for defective matrices — you can't diagonalize them.

Example: A = [2 1; 0 2] → λ = 2 (repeated) Only one eigenvector: [1; 0] — NOT diagonalizable!
Lecture 17

Diagonalization of Matrices

The Big Idea: A = SΛS⁻¹

If A has n independent eigenvectors, you can write:

A = S Λ S⁻¹ where: S = matrix of eigenvectors as columns [v₁ | v₂ | ... | vₙ] Λ = diagonal matrix of eigenvalues diag(λ₁, λ₂, ..., λₙ) S⁻¹ = inverse of the eigenvector matrix

This works because: AS = SΛ — each column of AS equals each column of SΛ.

When Can You Diagonalize?

  • ✓A must have n linearly independent eigenvectors
  • ✓S must be invertible (its columns must be independent)
  • ✗Two identical eigenvectors → two identical columns in S → S not invertible → cannot diagonalize
  • ✓Matrices with n distinct eigenvalues are always diagonalizable

Powers of A Made Easy

Diagonalization shines when computing Aᵏ:

Aᵏ = S Λᵏ S⁻¹ And since Λ is diagonal: Λᵏ = diag(λ₁ᵏ, λ₂ᵏ, ..., λₙᵏ)

This means raising a matrix to the 100th power just means raising each eigenvalue to the 100th power!

★ Generating Sequence Terms (e.g. Fibonacci)

Any linear recurrence uₙ₊₁ = Auₙ has solution uₙ = Aⁿu₀. With diagonalization:

  1. Write the recurrence as a matrix equation: e.g. Fibonacci → u = [Fₙ₊₁; Fₙ], A = [1 1; 1 0]
  2. Find eigenvalues λ₁, λ₂ of A (for Fibonacci: φ = (1+√5)/2 and ψ = (1−√5)/2)
  3. Find eigenvectors, build S and Λ
  4. Express u₀ = S·c (solve for constants c₁, c₂)
  5. uₙ = c₁λ₁ⁿv₁ + c₂λ₂ⁿv₂
Fibonacci closed form: Fₙ = (φⁿ − ψⁿ) / √5 where φ = (1+√5)/2 ≈ 1.618 (golden ratio) ψ = (1−√5)/2 ≈ −0.618
⚡ Exam Pattern

Your exam is likely to give a 2×2 matrix and a recurrence relation. Set up uₙ = Aⁿu₀, diagonalize A, and express the nth term using eigenvalues. The answer always takes the form c₁λ₁ⁿ + c₂λ₂ⁿ.

Worked Mini-Example

Find a formula for the nth term of: aₙ = 3aₙ₋₁ − 2aₙ₋₂, with a₀ = 0, a₁ = 1

Matrix form: [aₙ₊₁] [3 -2] [aₙ ] [aₙ ] = [1 0] [aₙ₋₁] Eigenvalues: λ² - 3λ + 2 = 0 → λ₁=2, λ₂=1 Eigenvectors: v₁=[2;1], v₂=[1;1] u₀ = [1;0] = c₁[2;1] + c₂[1;1] → c₁=1, c₂=-1 ∴ aₙ = 1·2ⁿ + (-1)·1ⁿ = 2ⁿ − 1
Lecture 18

Symmetric Matrices & SVD

Symmetric Matrices: Aᵀ = A

Symmetric matrices have three beautiful special properties:

  • 1All eigenvalues are real (even if you'd expect complex)
  • 2Eigenvectors for different eigenvalues are orthogonal
  • 3Always has n independent eigenvectors — always diagonalizable
⚡ Proof: Eigenvalues are Real

Suppose Ax = λx. Take conjugate transpose: x̄ᵀAᵀ = λ̄x̄ᵀ. Since Aᵀ=A, multiply both sides: x̄ᵀAx = λ̄(x̄ᵀx). But also x̄ᵀAx = λ(x̄ᵀx). Since x̄ᵀx = ||x||² > 0, we get λ = λ̄, so λ must be real.

Spectral Theorem

Every real symmetric matrix can be decomposed as:

A = Q Λ Qᵀ where Q is an ORTHOGONAL matrix of eigenvectors (Qᵀ = Q⁻¹) and Λ = diag(λ₁, λ₂, ..., λₙ)

This is the spectral decomposition. Expanding it gives:

A = λ₁q₁q₁ᵀ + λ₂q₂q₂ᵀ + ... + λₙqₙqₙᵀ

Each term qᵢqᵢᵀ is a projection matrix onto the direction of qᵢ. So every symmetric matrix is literally a weighted sum of projection matrices!

⚡ Proof Sketch: Spectral Theorem

Since A is symmetric, eigenvectors for distinct eigenvalues are orthogonal (provable by: if Av₁=λ₁v₁ and Av₂=λ₂v₂, then v₁ᵀAv₂ = λ₂(v₁ᵀv₂) = λ₁(v₁ᵀv₂). Since λ₁≠λ₂, we must have v₁ᵀv₂=0). Normalize them → Q is orthogonal → A = QΛQᵀ.

Positive Definite Symmetric Matrices

A symmetric matrix A is positive definite iff any of these equivalent conditions hold:

Eigenvalue Test
All λᵢ > 0
Pivot Test
All pivots > 0
Energy Test
xᵀAx > 0 for all x ≠ 0
Determinant Test
All upper-left subdeterminants > 0
⚡ Key Insight: Eigenvalues ↔ Pivots

For a symmetric matrix, the signs of the eigenvalues match the signs of the pivots. (They're generally not equal in value, but their signs agree.) This is the connection: positive definite ⟺ all eigenvalues positive ⟺ all pivots positive.

SVD — Singular Value Decomposition

SVD works for any matrix (even rectangular m×n). It generalizes diagonalization:

A = U Σ Vᵀ U = m×m orthogonal matrix (left singular vectors) Σ = m×n diagonal matrix (singular values σ₁ ≥ σ₂ ≥ ... ≥ 0) V = n×n orthogonal matrix (right singular vectors)

How to Compute SVD — Step by Step

  1. Compute AᵀA (an n×n symmetric, positive semidefinite matrix)
  2. Find eigenvalues of AᵀA: these are σᵢ² (squared singular values)
  3. The eigenvectors of AᵀA are the columns of V
  4. Compute singular values: σᵢ = √λᵢ
  5. Find U columns: uᵢ = Avᵢ / σᵢ
  6. Assemble: A = UΣVᵀ
Key relationships: AᵀA = V Σᵀ Uᵀ U Σ Vᵀ = V Σ² Vᵀ ← eigendecomp of AᵀA AAᵀ = U Σ Vᵀ V Σᵀ Uᵀ = U Σ² Uᵀ ← eigendecomp of AAᵀ

Geometry of SVD

SVD says every linear transformation A does 3 things in sequence:

Input → [Vᵀ: rotate/reflect] → [Σ: stretch along axes] → [U: rotate/reflect] → Output

The singular values σᵢ tell you the stretching factors along each principal direction. The larger σ₁ is relative to σ₂, the more "squished" or rank-one the transformation is.

⚡ Why SVD is Powerful

SVD reveals the true geometry of any matrix. The singular values measure the "energy" in each direction. Truncating to the top k singular values gives the best rank-k approximation — this is the math behind image compression, PCA, and recommendation systems.

Bonus

Conceptual Curveballs (Bonus Marks)

Sir likes these. They look simple but need deep understanding. Here are the most likely types:

★ "What happens to the projection matrix if A is scaled?"

From the quiz: if A is replaced by mA, does the projection matrix P change?

P = A(AᵀA)⁻¹Aᵀ P' = (mA)((mA)ᵀ(mA))⁻¹(mA)ᵀ = (mA)(m²AᵀA)⁻¹(mAᵀ) = m · (AᵀA)⁻¹ · m⁻² · mAᵀ ... = A(AᵀA)⁻¹Aᵀ = P ✓ UNCHANGED

Answer: The projection matrix is invariant to scaling of the subspace. Intuitively, projecting onto a line is projecting onto a line — stretching the line doesn't change which direction you're projecting onto.

★ "Can a matrix have the same eigenvalues as another but be different?"

Yes! Two matrices can share eigenvalues but have different eigenvectors — they're called similar matrices (related by A = PBP⁻¹). Eigenvalues are a property of the transformation, not the representation.

⚡ Deep Insight

The characteristic polynomial det(A − λI) only depends on the linear transformation, not the basis. Similar matrices have the same characteristic polynomial → same eigenvalues. But they can act completely differently on vectors.

★ "Is every diagonalizable matrix symmetric?"

No. Symmetry → diagonalizable (over reals, with orthogonal eigenvectors). But diagonalizable does NOT imply symmetric. Example: [1 1; 0 2] is diagonalizable (distinct eigenvalues) but not symmetric.

The special thing about symmetric matrices is that their eigenvectors can be chosen to be orthonormal — non-symmetric diagonalizable matrices generally don't have this.

★ "Why does SVD always exist but eigendecomposition doesn't?"

Eigendecomposition A = SΛS⁻¹ requires A to be square AND have n independent eigenvectors. SVD works because:

  • →AᵀA is always symmetric positive semidefinite → always has real, non-negative eigenvalues → singular values always real and non-negative
  • →The U and V are built from eigenvectors of symmetric matrices → always orthogonal
  • →Works for rectangular matrices — eigendecomposition requires square

★ "What does it mean geometrically if λ < 0?"

A negative eigenvalue means the corresponding eigenvector gets flipped — it points in the opposite direction after the transformation. The magnitude |λ| still tells you how much it stretches/shrinks, but the direction reverses.

Example: Reflection matrix has λ = −1 for the normal direction. Multiplying R by the normal vector gives you the negative of that vector.

★ "What is the relationship between eigenvalues and the determinant/trace?"

For A with eigenvalues λ₁, λ₂, ..., λₙ: det(A) = λ₁ · λ₂ · ... · λₙ (product of all eigenvalues) trace(A) = λ₁ + λ₂ + ... + λₙ (sum of all eigenvalues) These come directly from expanding det(A − λI): λⁿ − trace(A)λⁿ⁻¹ + ... + (−1)ⁿdet(A) = 0

If any eigenvalue is 0, the determinant is 0 → matrix is singular. This is why eigenvalues and invertibility are deeply connected.

★ "Every symmetric matrix is a combination of projection matrices — explain."

From the Spectral Theorem: A = λ₁q₁q₁ᵀ + λ₂q₂q₂ᵀ + ...

Each qᵢqᵢᵀ is the projection matrix onto the 1D subspace spanned by qᵢ. These projections are orthogonal to each other (the qᵢ are orthonormal). So A literally IS a weighted sum of orthogonal rank-1 projectors, with weights equal to the eigenvalues.

This gives a beautiful geometric picture: A stretches space by λᵢ in the direction of each eigenvector qᵢ.

Common Mistakes to Avoid

  • ✗Forgetting to check if S is invertible before diagonalizing
  • ✗Using eigendecomposition for a rectangular matrix (use SVD instead)
  • ✗Confusing σ (singular values) with eigenvalues — σ = √(eigenvalue of AᵀA)
  • ✗Not normalizing eigenvectors when building Q for symmetric matrices
  • ✗Forgetting: projection matrix eigenvalues are only 0 and 1
